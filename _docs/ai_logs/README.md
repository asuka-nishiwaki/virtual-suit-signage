# AI との対話ログ

提出用の AI ログはこのフォルダに保存してください。

## 保存方法（いずれか）

1. **共有リンク** — チャットの共有 URL を記載した `.txt` ファイルを置く
2. **Markdown / JSON** — やり取りの全文、または要約を `.md` / `.json` で保存

## ファイル例

```
_docs/ai_logs/
├── cursor_session_2026-05-24.md
└── chatgpt_share_link.txt
```

## 記載例（テキストファイル）

```
AI ツール: Cursor (Claude)
共有リンク: （ここに URL を貼る）
対象: バーチャルフィッティング プロトタイプ実装
```
